Phase 4: The Saboteur

The "First Word" Bias: If you see RA_N, you might immediately think "N" for RAN. But then you check PA_N and get PAN (Correct), and finally MA_N and get MAN (Correct).

The Fix: Always check if there is a better fit. If you used "I", you get RAIN, PAIN, and MAIN. Both "N" and "I" might work for one word, but usually only one letter creates a cohesive set of the same "type" (e.g., all 4-letter common nouns).
