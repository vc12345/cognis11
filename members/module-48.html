Phase 4: The Saboteur

The Trap: The "Face Count" Deception. Picking a net because it has 5 triangles, even though the solid only has 4.

The Fix: The Base-Edge Rule. Count the edges of the base (e.g., a square has 4). This number is exactly how many side-flaps must exist. If the base is a triangle, you only need 3 flaps.
