Module 29: Alpha-Numeric Decoder

Phase 1: The X-Ray

Alpha-Numeric decoding is the process of translating linguistic symbols into mathematical values. It requires Mapping Consistency: whether the code is a direct sequence (A=1), a reversed sequence (A=26), or a shifted sequence (A=3), the logic must be applied uniformly across the entire string without breaking the pattern.

Advanced Year 2 "If A=1 and B=2, what is the numerical code for the word 'FISH'?"
Average Year 4 "If the alphabet is reversed so A=26 and Z=1, what is the code for 'CAT'?"
Advanced Year 5 "A secret code uses a +3 offset (A=4). Decode the sequence '4-18-21-13' to find the word."

Phase 4: The Saboteur

The "Wrap-Around" Trap: When a shift moves past Z (26) or before A (1), students often get confused and stop. For example, in a -2 shift, they might think A has no value.

The Fix: Use Circular Logic. Treat the alphabet like a clock face. If you are at Z and go forward 1, you land back on A. If you are at A and go back 1, you land on Z. The sequence 1-26 is a continuous loop that never ends.
