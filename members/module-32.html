Module 32: Symbol-to-Word Mapping

Phase 1: The X-Ray

Breaking a symbol code is about spotting Patterns. If a symbol repeats, the letter it hides must also repeat in that exact spot. By matching the "shape" of the code (where the duplicates are) to the "shape" of the words, you can crack the entire message without needing a key.

Advanced Year 2 "If Triangle = G and Circle = O, what 3-letter word is Triangle-Circle-Triangle?"
Average Year 4 "The word is either STEER, STARE, or STAIR. If the code is Square-Bolt-Star-Star-Leaf, which word is it? How do you know?"
Advanced Year 5 "Given the code: Star-Circle-Bolt / Star-Leaf-Sun. If the words are 'THE END' and 'THE DOG', find the code for 'G'."

Phase 4: The Saboteur

The "Visual Mimic" Trap: Trying to "guess" letters based on what a symbol looks like (e.g., assuming a Circle must be 'O' or a Bolt must be 'S').

The Fix: Ignore the shape; follow the Position. If a symbol appears at positions 2 and 3 of a 5-letter word, you are looking for words with double letters in the middle (like APPLE or SPEED). Logic beats eyes every time.
