Audit: Counting heights per coordinate.

Phase 4: The Saboteur

The Trap: Counting visible faces. A single corner block can show 3 faces, making you count it as 3 blocks.

The Fix: The Column Tally. Pick the top-most face in a column. Note its height (e.g. 3 blocks). Move to the next. Summing column heights prevents double-counting.
