Module 4: Inverse Proportion

THE "INVARIANT WORK" PRINCIPLE

Phase 1: The X-Ray

A "System 1" brain expects things to be linear: "If I double the workers, I double the days." But common sense says more workers = less time. The logic is inverted.

Phase 2: The Anatomy

Think of work as a rectangle. The Area (Total Work) stays the same. If the width (workers) gets bigger, the height (days) must get shorter to keep the area constant.

The Constant: Workers × Days = Total Work Units.
If you change the workers, you must divide that same Total Work by the new number of workers.
