Module 39: Relationship Analogies

Phase 1: The X-Ray

An analogy is a Logical Ratio. The bridge that connects the first pair must be the exact same bridge that connects the second. To crack the code, you must build a "Bridge Sentence" (e.g., A [Verb] B). If the sentence works for the first pair, it must be used as a rigid template for the second to find the missing word.

Phase 4: The Saboteur

The "Inversion" Trap: Getting the relationship correct but the order wrong. If the first pair is Subject : Property, the second pair cannot be Property : Subject.

The Fix: Use the Sentence Template. If you say "A Hand HAS A Finger," you must test the second pair using the same order: "A Foot HAS A Toe." If you say "A Toe HAS A Foot," the logic is inverted and your answer will be wrong. Direction is as important as the connection itself.
