Module 45: Attribute Matrices

Phase 1: The X-Ray

A matrix is a grid where items change based on row or column rules. To solve for the missing cell, you must find the "Constant" (what stays the same) and the "Variable" (what changes). If color changes across rows but shape changes down columns, the final cell is a fusion of both rules.

Phase 4: The Saboteur

The Trap: The "One-Rule" Trap. Students often find one pattern (e.g., the shape is a circle) and pick an answer immediately, ignoring that the color or orientation might follow a different rule.

The Fix: The Cross-Hair Method. Solve the Row first (to find Attribute A), then solve the Column (to find Attribute B). The answer must satisfy the intersection of both rules. If you only look horizontally, you're only seeing half the logic.
