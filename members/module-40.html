Module 40: Odd One Out (Semantic)

Phase 1: The X-Ray

Odd One Out questions test classification. Success depends on identifying a Rule of Inclusion. If four words are "Mammals" and one is a "Bird," the bird is the outlier. The deeper the rule, the harder the question.

Phase 4: The Saboteur

The "Double Identity" Trap: Choosing a word because it's "different" in a very simple way (e.g., "it starts with the letter S") while ignoring a much stronger category rule.

The Fix: Always verify the Majority Rule. If you think SPATULA is the odd one out because it has 7 letters, check if the other four have the same number of letters. If they don't, your rule is too weak. A true outlier is excluded by a rule that 100% of the others obey.
