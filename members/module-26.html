Module 26: Rotational Symmetry

Phase 1: The X-Ray

Rotational symmetry is about how many times a shape looks exactly the same during one full $360^\circ$ turn.

Phase 2: The Anatomy

The "Order" is the number of times it fits. A square has order 4. A rectangle has order 2.

Phase 4: The Saboteur

The "Order 0" Trap: There is no such thing as Order 0. Every shape looks like itself after $360^\circ$, so the minimum order is 1.
