Ranking items is about Relativity. You are placing pieces on a mental line, expanding the space between them as new clues arrive. Don't look for the "winner" first; look for the "neighbor."

Phase 4: The Saboteur

The Trap: Storing the entire list in your short-term memory without a visual anchor. Memory is easily "overwritten" when clues are complex.

The Fix: The Vertical Ladder. Always sketch a line. If A > B, draw A above B. If C > A, draw C above A. This physical spacing makes the final rank obvious.
