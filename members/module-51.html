Module 51: Block Counting

Phase 1: The X-Ray

Block counting is a test of Volumetric Inference. In a 2D isometric drawing, many blocks are partially or completely hidden from view. To calculate the total volume, you must apply the "Support Logic": if a block is sitting at height $Z$, there must be exactly $Z-1$ blocks stacked beneath it to act as a foundation. Counting only visible faces is the most common path to failure.

Phase 4: The Saboteur

The "Face-Value" Trap: Counting visible diamond faces instead of whole blocks. A single corner block on the top level can show 3 faces, tricking your brain into counting it three times.

The Fix: Use Column Tallying. Pick one top-facing diamond. Identify its height (e.g., it is on the 3rd floor). That represents a column of 3 blocks. Note the number, move to the next top-facing diamond, and sum the heights. This ensures you count every foundation block exactly once.
