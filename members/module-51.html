Phase 4: The Saboteur

The Trap: Counting visible faces. A single corner block can show 3 faces (top, front, side), leading students to count it as 3 separate blocks.

The Fix: The Top-Down Tally. Pick a column and find the top-most block. Note its height (e.g., 3 blocks high). Repeat for all columns and add the numbers. This ensures you count every hidden support block without double-counting visible faces.
