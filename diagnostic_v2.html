Profile Type A

The Deep Understander

✓
Prioritises Logic Over Rules
Wants to understand why a method works rather than just following the steps.
✓
Pauses Before Execution
Deliberately waits to fully comprehend the problem before taking action.
✓
Time-Pressure Sensitive
Accuracy drops noticeably when a clock is running, despite knowing the material.
✓
Resilient on Novel Problems
Often outperforms peers on unusual questions that cannot be pattern-matched.
✓
Explicit Reasoning Profile
Can explain their thought process step-by-step with high clarity.
✓
Repetition Fatigue
Visibly loses focus or becomes bored after 8 or 9 of the same question type.
✓
Knowledge Gap Persistence
Will dwell on a single unknown word or detail until it is logically resolved.
✓
Rejection of Shortcuts
Reluctant to use "tips" or "tricks" that haven't been logically explained.
✓
Holistic Perspective
Needs to see how a concept fits into the "whole picture" before starting.
✓
Forensic Self-Correction
Only satisfied when they find the underlying reason for a mistake.
✓
Anti-Guessing Bias
Unwilling to guess an answer, even when time is running out.
✓
Reconstruction Instinct
Tends to re-derive the logic of a solution every time they see it.
Profile Type B

The Pattern Recogniser

✓
Action-First Default
Jumps into problems immediately, often before finishing the reading.
✓
Intuitive Pattern Matching
Confidently spots connections but struggles to explain the "how".
✓
Natural Timed Pace
Speed comes naturally; rarely feels stressed by a running clock.
✓
High Drill Efficiency
Scores improve steadily and reliably with high-volume repetitive practice.
✓
High Correction Tolerance
Accepts a corrected answer and moves on without needing a deep explanation.
✓
Impatience with Planning
Restless when asked to plan or reflect; prefers to start doing.
✓
Friction on Novelty
Struggles with questions phrased in unfamiliar ways, despite the difficulty level.
✓
Trial-and-Error Learning
Comfortable learning the mechanics of a task through action rather than theory.
✓
Context Filtering
Efficiently ignores small, irrelevant details to get to the answer faster.
✓
Answer-First Mindset
Often says they "just knew" the result without being able to name the rule.
✓
Competitive Speed
Thrives in fast-paced environments; motivated by finishing first.
✓
Structural Dependency
Works best when question types are clearly categorized and familiar.
